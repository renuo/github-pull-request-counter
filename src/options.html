GitHub Pull Request Counter

developed by Renuo

Counter

Customize which required actions increase the counter.

Maximum Age

Customize the maximum age of pull requests (in days). It does not apply to "I must review" and "My team must review."

Scope

Enter a comma seperated list of GitHub account names from which the extension should fetch. (case insensitive)

Teams

Enter a comma seperated list of GitHub teams of which the team review requests you would like to be displayed seperatly. If a team is not present here it's review requests will be displayed in "I must review".

Ignored PRs

Below you can enter a comma seperated list of PRs you would like to ignore. Redundant PRs will be removed from the counter automatically, but will still be displayed in the popup.

Ignored titles

You can ignore PRs that include specific text in titles. Use a regular expression to match the title. They will not be counted in the counter, but will still be displayed in the popup.

Access Token

Generate a personal access token from here . Leave the permissions as is and copy the token in the text box below.
